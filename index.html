Cardiovascular Disease Prediction Model

This ongoing project aims to predict cardiovascular disease using a dataset with features like age, gender, blood pressure, cholesterol levels, BMI, and lifestyle habits. The model preprocesses data with imputation, scaling, and encoding while incorporating engineered features like age ranges by gender. Using a Random Forest classifier with hyperparameter tuning and class imbalance handling, the goal is to create a robust, balanced model to identify high-risk individuals. This work has the potential to aid in early diagnosis and optimize healthcare resources.
